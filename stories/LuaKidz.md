# LuaKidz

LuaKidz gamifies Roblox Lua learning for 8-12 year olds with bite-sized quests, AI hints, and parent tracking.[web:93][web:94]

## Product Vision
Kids love Roblox but Lua barriers stop creation; structured fun lessons build coding confidence.[web:98]

## Target Users
- Parents seeking safe coding intro (ages 8-12)
- Teachers for classroom Roblox projects
- Kids wanting to make obbies/tycoons

## Problem Statement
Classes exist but self-paced gamified apps lacking; Studio overwhelming alone.[web:93][web:97]

## Core Value Proposition
Master Lua via Roblox quests. Badges, AI help, publish-ready games in weeks.

## Functional Requirements

### Quest Lessons
- 50+ levels: variables to events
- Obbie/tycoon projects
- Visual scripting to Lua

### AI Tutor
- Code hints/debug
- Natural chat help
- Progress-adaptive difficulty

### Parent/Teacher Dashboard
- Weekly reports
- Screen time limits
- Shareable projects

### Roblox Integration
- Export to Studio
- Playtest in-app
- Publish button

## User Experience
- Kid-friendly UI
- 15min sessions
- Voice narration

## Constraints
- Lua basics-events
- Roblox-safe only
- English

## Success Metrics
- 80% lesson completion
- 1K games published
- 4.8 stars

## Monetization
- Free first 10 quests
- $4.99/mo full access
- School $99/yr/class

Fills self-paced gap vs classes.[web:94]
