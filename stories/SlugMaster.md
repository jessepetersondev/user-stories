# SlugMaster

SlugMaster generates unique SEO slugs from title lists in one click.[web:117]

## Product Vision
CMS users manually slugify; bulk tool saves time.

## Target Users
Bloggers, marketers bulk posting.

## Why Exists
No free bulk slugifier with dupes handled.

## Core Feature
Paste title list → numbered slugs → copy all.

## Requirements
Input textarea. Output list. Copy button.

## Constraints
Titles only. Client-side. No save.

## Success Criteria
5K visits/mo. Pro $1/mo history.

Monetization: Free basic, pro batch 1000+.
