# Base64Image

Base64Image converts images to copyable Base64 strings for devs embedding in HTML/CSS/JSON.[web:126]

## Product Vision
Devs manually convert images; one-click solves.

## Target Users
Frontend devs, email marketers embedding images.

## Why Exists
No dead-simple web tool with copy button.

## Core Feature
Upload image → instant Base64 → copy button.

## Requirements
Drag-drop file. Show preview/string. Copy to clipboard. Dark mode toggle.

## Constraints
Images only. 10MB max. No accounts.

## Success Criteria
10K mo visits. Pro $2/mo unlimited size.

Monetization: Free 5MB, pro unlimited.
